Technical Specifications
-------

* Emission: ~ 31,000,000 KPL
* Premine: 2%
* Mining Algorithms: NeoScrypt, Argon2-4096, LyraCZ
* Block time: 2 minutes (6 minutes per algorithm)
* **Ports:**
*  RPC: 12101 
*  P2P: 12102
*  Block halving: Every 4 years
